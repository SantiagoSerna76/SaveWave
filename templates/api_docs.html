API de SaveWave

Integra descargas de video y audio MP3 en tus propias aplicaciones, bots o scripts.

Exclusivo plan Premium

Autenticación

Todas las peticiones a la API requieren un Token JWT en el header Authorization.

Cómo obtener tu token:

  1. Inicia sesión en tu cuenta Premium
  2. Ve a tu Panel de control
  3. Haz clic en "Generar Token API"
  4. Copia el token generado (válido por 30 días)

Header requerido en cada petición:

Authorization: Bearer TU_TOKEN_AQUI

URL Base

https://savewave.software

POST /api/video-info

Obtiene metadatos de un video (título, duración, calidades disponibles, thumbnail) sin descargarlo.

Parámetros (JSON body)
ParámetroTipoRequeridoDescripción
urlstringSíURL del video (YouTube, TikTok, Instagram, etc.)
Ejemplo con cURL
curl -X POST https://savewave.software/api/video-info \
  -H "Authorization: Bearer TU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"}'
Ejemplo con Python
import requests

headers = {
    "Authorization": "Bearer TU_TOKEN",
    "Content-Type": "application/json"
}

response = requests.post(
    "https://savewave.software/api/video-info",
    json={"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"},
    headers=headers
)

data = response.json()
print(data["title"])           # Título del video
print(data["duration"])        # Duración en segundos
print(data["available_qualities"])  # ['360p', '720p', '1080p']
Respuesta exitosa

POST /api/download

Descarga un video y devuelve un enlace temporal para descargarlo.

Parámetros (JSON body)
ParámetroTipoRequeridoDescripción
urlstringSíURL del video
qualitystringNoCalidad: 360p, 720p, 1080p, 2160p (default: 720p)
Ejemplo con cURL
curl -X POST https://savewave.software/api/download \
  -H "Authorization: Bearer TU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ", "quality": "1080p"}'
Respuesta exitosa
{
  "success": true,
  "filename": "video_1719817200_titulo.mp4",
  "file_size": "15.20 MB",
  "title": "Nombre del video",
  "platform": "youtube",
  "download_url": "https://savewave.software/downloads/video_1719817200_titulo.mp4"
}

POST /api/download-audio

Extrae y descarga el audio de un video en formato MP3.

Parámetros (JSON body)
ParámetroTipoRequeridoDescripción
urlstringSíURL del video
qualitystringNoCalidad: 128 o 320 kbps (default: 320 para Premium)
Ejemplo con Python
import requests

headers = {
    "Authorization": "Bearer TU_TOKEN",
    "Content-Type": "application/json"
}

# Descargar como MP3
response = requests.post(
    "https://savewave.software/api/download-audio",
    json={"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"},
    headers=headers
)

data = response.json()
if data["success"]:
    # Descargar el archivo MP3
    mp3 = requests.get(data["download_url"])
    with open("cancion.mp3", "wb") as f:
        f.write(mp3.content)
    print(f"Descargado: {data['title']} ({data['file_size']})")

POST /api/auth/login

Alternativa: obtener un token JWT programáticamente usando tus credenciales.

curl -X POST https://savewave.software/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "tu@email.com", "password": "tu_password"}'
